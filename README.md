# Joe Faith — personal website

A fast, responsive static portfolio for GitHub Pages.

## Publish

1. Create a GitHub repository and push these files to its `main` branch.
2. In the repository, open **Settings → Pages**.
3. Under **Build and deployment**, choose **GitHub Actions**.
4. The included workflow will publish the site automatically.

To update the site, edit `index.html`. Every push to `main` triggers a fresh deployment.
